## Union Pharma

This repository contains the **Union Pharma** website, a static web project developed using **HTML** and **CSS**. The website is designed for a pharmaceutical company, providing a professional and user-friendly online presence to showcase their products and services.

### Key Features:

- **Responsive Design:**
  - The website is built with a responsive design approach, ensuring optimal viewing and interaction across a variety of devices and screen sizes.

- **Product Showcase:**
  - Features sections to display pharmaceutical products with detailed descriptions, images, and specifications.

- **Service Information:**
  - Dedicated sections outlining the services offered by Union Pharma, including details about their expertise and offerings.

- **Contact Form:**
  - Includes a contact form allowing users to get in touch with the company for inquiries, feedback, or support.

- **Navigation and Layout:**
  - Clear and intuitive navigation for easy access to different sections of the site. The layout is designed to be user-friendly and visually appealing.

### Tech Stack:

- **Frontend:** HTML, CSS
